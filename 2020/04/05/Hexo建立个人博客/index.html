Hexo建立个人博客小白教程：

title: Hexo建立仆人博客小白教程
date: 2020-03-20 21:46:15
tags: 本教程在windows7操作系统下完成，其他操作系统请适当参照：

1. 下载安装node.js

node.js是安装hexo的必备工具之一，官方网站下载地址是：nodejs.org/en/

安装过程基本没什么坑，就不啰嗦了。

2. 安装cnpm

由于npm服务器在国外，访问时速度很慢，所以用镜像安装cnpm，在cmd根目录下输入以下命令

npm install -g cnpm --registry=https://registry.npm.taobao.org

3.安装Hexo

cnpm install -g hexo-cli

4.新建文件夹

在电脑里新建一个文件夹用来存放博客或个人站点的文件夹，并在cmd里进入该文件夹，比如我建的是位于F盘下的Blog文件夹，cmd命令如下：

f:
cd blog

5.初始化blog文件夹

hexo init

6.启动文件夹

hexo s

7.新建.md文件

进入blog\source_posts文件夹下，新建一个.md文件，

cd source\_posts
hexo n "新文件名"

注意：这里有个坑，我遇到过问题，一般时，这个.md文件的头部应该有如下信息

---
title: 文件名
date: YYYY-MM-DD HH:MM:SS
tags: 描述（可不写）

---

8.启动并预览博客

//先返回到blog目录下
cd../..
//再输入下面的命令
hexo g
//生成结束后，可以预览网页，hexo会生成一个默认的theme和一篇hello world文章
hexo s
//这时会生成一个localhost:4000端口的网页，可以预览
//预览结束满意后，在cmd里按ctrl+c\y\enter结束

9.安装deployer

cnpm install --save hexo-deployer-git

10.在个人github主页里新建一个项目

命名为：github个人用户名.github.io
//切记：名字一定不能错
//然后复制地址备用

![建站4 (1)](D:\仰望星空\Github\image\建站4 (1).jpg)

11.修改blog系统配置文件

在blog文件夹下找到_config.yml文件，用记事本打开，把以下信息修改成个人信息

github个人用户名.github.io
    type: 'git'
    repo: 'http://github.com/github个人用户名/github个人用户名.github.io.git'
    branch: 'master'

12.向github推送

//cmd下输入以下命令
hexo d

这时，就大功告成了！

如果出现错误，随时可以把blog文件夹删掉，重新来一遍就行了。

剩下的事情就是向博客里放文章和美化主题等等工作，就不是能帮你的了。

祝你好运！

我的博客地址是：https://xyu100.github.io/，快来看看吧！
